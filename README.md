# My Hyprland Rice Setup

**Disclaimer:** This configuration works perfectly on my machine, but it might not work out of the box on yours. Please review the configurations before applying them to your own system.

To clarify, I didn't develop this from the ground up. I just modified a deprecated Hyprland configuration to make it compatible with my system :)

## My Hardware Specifications

- **OS:** CachyOS Linux (x86_64)
- **CPU:** 11th Gen Intel Core i5-1145G7 @ 2.60GHz (8 Cores)
- **RAM:** 32 GB
- **Host:** ThinkPad T14 Gen 2 (T-14g2)

## Keybindings Documentation

The main modifier key is **SUPER**.

### Application Shortcuts

- `Super + T` : Open Terminal (Kitty)
- `Super + E` : Open File Manager (Dolphin)
- `Super + F` : Open Web Browser (Firefox)
- `Super + C` : Open EWW Control Center
- `Super + Shift + C` : Open VSCode
- `Super + M` : Launch Custom Music Layout (Spotify, Cava, Sptlrx, Terminal-rain, Tty-clock)
- `Super + A` : Launch App Launcher (Rofi)
- `Super + Tab` : Switch between open applications (Rofi)
- `Super + R` : Browse system files (Rofi)

### System & UI Controls

- `Super + Q` : Close active window
- `Super + W` : Toggle floating window
- `Super + L` : Lock Screen
- `Super + Backspace` : Logout Menu
- `Super + B` : Toggle between Waybar and EWW
- `Ctrl + Escape` : Toggle Waybar on/off
- `Ctrl + Shift + Escape` : Open System Monitor
- `Super + P` : Snip an area for screenshot
- `Super + Shift + W` : Open Wallpaper Selection Menu (Rofi)
- `Super + V` : Open Clipboard Manager (Cliphist)

### Workspaces & Windows

- `Super + [1-0]` : Switch to workspace 1-10
- `Super + Shift + [1-0]` : Move active window to workspace 1-10
- `Super + Arrow Keys` : Move focus between windows
- `Super + Shift + Arrow Keys` : Resize active window
- `Super + Shift + Ctrl + Arrow Keys` : Move active window position
- `Super + Mouse Left Click (Hold)` : Move floating window
- `Super + Mouse Right Click (Hold)` : Resize floating window
